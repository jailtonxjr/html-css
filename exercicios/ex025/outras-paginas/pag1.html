Testando os conhecimentos da aula 8

Em resumo, pode se destacar o uso das tag's <strong>, para deixarmos algum texto em negrito, bem como, podemos usar a tag <em>, para deixarmos algum texto em itálico.

Os texto marcados são representados pela tag <mark>, por padrão, a cor de marcação é destacada pelo AMARELO, mas podendo ser altarada atráves do CSS.

Logo após, aprendemos a aplicar as tags referentes a textos pequenos, deletados e inseridos. Sendo elas: <small> = textos pequenos; <del> = textos deletados;<ins> = textos inseridos

Por fim, vimos como usar as tags de representação para textos sobrescrito e subscritos, sendo elas: <sup> = sobrescrito; <sub> = subscrito

No mais é isso, até a próxima!
