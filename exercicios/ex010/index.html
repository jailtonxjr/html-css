Criando links externos

Para criamos links externos usamos a tag <a> juntamente com os atributos href="" na tag de abertura e entre as tags o espaço clicável, como esse espaço para acessar o youtube

Com o target conseguimos abrir o link em uma nova janela em branco e usando o rel dizemos ao negador que o link é externo
