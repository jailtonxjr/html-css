Código

Nesta aula vimos o uso de várias tags no HTML, como estamos estudando um assunto voltado para tecnologia é certa a necessidade de expor código de programação ou de marcação em nossas páginas, logo, para esse fim, usamos a tag <code>, com ela conseguimos expor nossos códigos de forma semântica e com o texto mono-espaçado, facilitando sua leitura. Entretando, o nosso cósigo não será identado, logo, para resolvermos esse problemas, nós usamos a tag <pre> = pré-formatado, com ela nós conseguimos manter toda a estrutua do código digitada no HTML

n = int(input()) if n % 2 = 0 print(n, "é primo")

Usando a tag <pre> chegamos nesse resultado: 


    n = int(input())
    if n % 2 = 0
        print(n, "é primo")
